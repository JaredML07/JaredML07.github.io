¿Quién soy?

Hola, soy Jared Muñoz Lozano, estudiante de la escuela preparatoria Bachilleres Veracruz, del 5° grado, salón "B". Esta página fue diseñada como parte de un proyecto para mi maestro Luis Fuentes Carreto.

Objetivos del Proyecto

  • Crear un proyecto visual y educativo para el 3er parcial.
  • Concientizar sobre el cuidado del agua y su problemática.
  • Presentar una página atractiva y útil para mis compañeros.

Conclusiones

En conclusión, este proyecto no solo busca cumplir con un requisito académico, sino también generar conciencia sobre la importancia del agua, un recurso vital que debemos proteger. Espero que esta página inspire a otros a tomar medidas para cuidar este bien tan necesario para la vida.
